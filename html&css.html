1. Basic HTML Structure (Easy)

Problem: Create a basic HTML document with a DOCTYPE declaration, head, and body. Include a header, a main section with a paragraph, and a footer. 


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Basic HTML Structure</title>
  </head>
  <body>
    <header>
      <h1>Welcome to My Website</h1>
    </header>
    <main>
      <p>This is a simple HTML document.</p>
    </main>
    <footer>
      © 2025 My Website
    </footer>
  </body>
</html>

2. HTML Form (Slightly Medium)

Problem: Create an HTML form that collects a user's name, email, and message. Include proper labels, input fields, and a submit button. Use minimal CSS to style the form. 


<!-- HTML Form -->
<form action="#" method="post" class="contact-form">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required />
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required />
  
  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="4" required></textarea>
  
  <button type="submit">Submit</button>
</form>
        

/* CSS for the HTML Form */
.contact-form {
  max-width: 500px;
  margin: 20px auto;
  padding: 10px;
  border: 1px solid #ccc;
  border-radius: 4px;
}
.contact-form label {
  display: block;
  margin: 10px 0 5px;
}
.contact-form input,
.contact-form textarea {
  width: 100%;
  padding: 8px;
  margin-bottom: 10px;
}
.contact-form button {
  background: #35424a;
  color: #fff;
  border: none;
  padding: 10px 15px;
  cursor: pointer;
}
.contact-form button:hover {
  background: #2c3e50;
}

3. Responsive Navigation Bar (Medium)

Problem: Build a responsive navigation bar using HTML and CSS. The navigation should display horizontally on larger screens and stack vertically on smaller screens. 


<!-- Navigation Bar HTML -->
<nav class="navbar">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
        

/* CSS for Responsive Navigation Bar */
.navbar ul {
  list-style: none;
  display: flex;
  justify-content: center;
  background: #35424a;
  padding: 10px;
}
.navbar ul li {
  margin: 0 15px;
}
.navbar ul li a {
  color: #fff;
  text-decoration: none;
  font-weight: bold;
}

/* Responsive: Stack items on smaller screens */
@media (max-width: 600px) {
  .navbar ul {
    flex-direction: column;
    align-items: center;
  }
  .navbar ul li {
    margin: 10px 0;
  }
}

4. Multi-Column Layout (Medium-Hard)

Problem: Create a multi-column layout using Flexbox. The page should display three columns on wider screens and stack them on narrower screens. 


<!-- Multi-Column Layout HTML -->
<div class="flex-container">
  <div class="flex-item">Column 1</div>
  <div class="flex-item">Column 2</div>
  <div class="flex-item">Column 3</div>
</div>
        

/* CSS for Multi-Column Layout */
.flex-container {
  display: flex;
  gap: 15px;
  margin: 20px;
}
.flex-item {
  background: #f4f4f4;
  padding: 20px;
  flex: 1;
  border: 1px solid #ccc;
  text-align: center;
}

/* Responsive: Stack columns on small screens */
@media (max-width: 600px) {
  .flex-container {
    flex-direction: column;
  }
}

5. CSS Animations & Transitions (Hard)

Problem: Create a button with a hover effect using CSS transitions and keyframe animations. The button should change background color and scale slightly on hover, and the effect should be smooth. 


<!-- Animated Button HTML -->
<button class="animated-btn">Hover Me</button>
        

/* CSS for Animated Button */
.animated-btn {
  padding: 12px 20px;
  font-size: 16px;
  color: #fff;
  background: #35424a;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  transition: transform 0.3s ease, background 0.3s ease;
}

.animated-btn:hover {
  background: #2c3e50;
  transform: scale(1.05);
}

/* Optional: Keyframe animation example */
@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.08); }
  100% { transform: scale(1); }
}

/* Apply animation on focus (for example) */
.animated-btn:focus {
  outline: none;
  animation: pulse 1s infinite;
}
